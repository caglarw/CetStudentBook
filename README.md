In this project, I added the "Book" section to the main project. The functions of this book Book section is similarly with the previous (only Student model) version. 
The functions are Creating, Editing, Listing and Deleting the books to the system.
I used SQLite for handling the database part and done the connections and rearranged the migration folder.

In order to run the project, open the terminal in the project and run this code:
"dotnet ef database update" to create the database.
Then start the project on IDE, in order to start the app.
